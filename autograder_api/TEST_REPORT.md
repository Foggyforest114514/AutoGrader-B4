## 📝 运行命令

```bash
# 运行所有测试
python -m pytest tests/ -v

# 生成覆盖率报告
python -m pytest tests/ --cov=app --cov-report=html --cov-report=term

# 运行单个模块测试
python -m pytest tests/test_auth.py -v

# 并行执行测试（需要安装 pytest-xdist）
python -m pytest tests/ -n 4
```
